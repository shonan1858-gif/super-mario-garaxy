# Mini Galaxy Runner

ChatGPT共有設計を元にした、**マリオギャラクシー風MVP**（球面重力 + 接線移動 + ゴール判定）のブラウザ実装です。

## セットアップ

```bash
npm install
npm run dev
```

## 操作

- `WASD`: 移動（カメラ相対 / 接線平面）
- `Space`: ジャンプ（接地時のみ）
- マウス: カメラ回転（クリックでポインタロック）
- `R`: リスタート

## 実装済みMVP要素

- 球体惑星（半径50）
- 球面重力（惑星中心方向へ一定加速度）
- 局所Upに沿う接線移動
- 接地レイ判定 + ジャンプ
- 三人称カメラ（局所Up追従）
- ゴールリング接触でCLEAR表示
- 外周落下でリスポーン
- HUD（Speed / Grounded）

## 手動テスト項目

1. 起動後、惑星・プレイヤー・ゴールリングが表示される。
2. `WASD`で移動でき、惑星を回っても操作方向が破綻しない。
3. `Space`でジャンプでき、空中連打で多段ジャンプしない。
4. ゴールリングに触れると`CLEAR! Press R to Restart`が表示される。
5. 画面外へ落下した場合、自動で初期位置にリスポーンする。
6. `R`で即時リスタートできる。
